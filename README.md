# DriveX

DriveX is a distributed cloud storage platform built with Node.js, MongoDB, and React.

## Features

- JWT Authentication
- File Upload API
- MongoDB Metadata Storage
- Secure Protected Routes

## Tech Stack

Backend:
- Node.js
- Express
- MongoDB
- Multer

### MinIO Setup

Download MinIO:
https://min.io/download

Run:

minio server ./storage --console-address ":9001"

Console:
http://localhost:9001
